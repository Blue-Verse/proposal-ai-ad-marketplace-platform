핵심 기술Next.js 14 (App Router) + TypeScript + Tailwind + shadcn/ui + Supabase (Auth/DB/Storage/RLS) + Vercel + Resend + Sentry + 토스페이먼츠

핵심 요구사항 분석

  • 인증/회원 — 이메일 + 카카오 + 구글 SSO, 광고주/크리에이터 역할 구분, 마이페이지, 이메일 인증, 약관/정책 페이지(텍스트 발주처 제공)
  • 광고주 기능 — 미션 CRUD, 브랜드 에셋 업로드, IP 활용 옵션, 시안 목록/상세(영상 재생)/선정, 결제 요청 발행
  • 크리에이터 기능 — 미션 검색·카테고리 필터·정렬, 미션 상세(IP 정보), 시안 제출(영상 200MB·IP 명시·첨부), 내 제출 내역/선정 결과
  • 관리자 페이지 — 회원·미션·시안·IP 조회/수정, 시드 입력 도구, 신고/차단, 결제 입금 확인 후 미션 활성화, Slack/이메일 알림, 통계 대시보드
  • IP Showcase — IP 카탈로그(이름·썸네일·설명·사용 조건), 미션 등록 시 IP 연결, 시안 IP 뱃지, 관리자 수동 등록
  • 결제 링크 — 토스페이먼츠 결제 링크 또는 무통장입금 안내, 관리자 수동 입금 확인
  • 오픈베타 필수 — 신고/차단, 이메일 인증, rate limiting, 동일 IP 다중계정 차단, SEO/OG, Supabase RLS, Sentry
  • 알림/디자인 — Resend 이메일(시안 제출/선정/결제), Slack Webhook, Figma 시안 기반 반응형 퍼블리싱
  • 영상 처리 — Supabase Storage 직접 업로드(200MB, mp4/mov), 트래픽 증가 시 Mux/Cloudflare Stream 전환 가능한 추상화 레이어

제안 포인트

  • 다중 역할 RBAC를 RLS 정책으로 매핑 — 7 사용자 역할 운영 SaaS 경험을 본 프로젝트 3 역할(광고주/크리에이터/관리자) 교차 권한 매트릭스로 첫 주에 정의하여 후반 보안 리뷰 비용 절감
  • 영상 업로드 추상화 레이어 우선 설계 — Supabase Storage 직접 업로드를 인터페이스화하여 트래픽 증가 시 Mux·Cloudflare Stream으로 비파괴 전환 가능한 구조 보장
  • 결제 링크 + 관리자 수동 승인 워크플로우의 안정적 분리 — 결제 요청·입금 확인·미션 활성화를 명시적 상태 머신으로 모델링
  • 오픈베타 보안 항목을 인증 골격과 동시 셋업 — RLS·rate limiting·SSO·이메일 인증을 Phase 1 내 일괄 적용
  • 스코프 잠금 — IP Showcase는 베타 간소화 버전(카탈로그 + 미션 연결 + 시안 IP 뱃지)에 집중, 정산 시스템·자동 PG·실시간 채팅·AI 영상 생성은 Phase 2로 명시 분리
  • 인수인계 가능한 산출물 구조 — README, ERD, RLS 명세, API 명세, 환경변수, 배포 가이드를 코드와 동기화하여 중장기 다른 개발자 협업 보장

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

  • 광고주·크리에이터·관리자 3 사용자 그룹이 미션·시안·IP·결제를 중심으로 거래하는 챌린지형 3면 마켓플레이스
  • 2026년 7월 둘째 주 오픈 베타 런칭 — 일반 사용자 트래픽 대응 안정성·보안 확보가 목표
  • 차별점: IP Booster — 드라마/연예인/웹툰 IP를 광고 소재로 활용하는 라이선싱 레이어
  • 베타 단계 스코프: IP 정산, 자동 결제 풀, 실시간 채팅, AI 영상 생성 등은 Phase 2로 명시 제외

3. 기능 요구사항

3.1 인증 / 회원

  • [필수] 이메일 회원가입 + 이메일 인증(Resend)
  • [필수] 카카오·구글 SSO (Supabase Auth)
  • [필수] 광고주/크리에이터 역할 구분 및 가입 시 선택
  • [필수] 마이페이지(프로필 수정, 비밀번호 변경, 계정 삭제 요청)
  • [필수] 약관/개인정보처리방침/환불정책 페이지(텍스트 발주처 제공)

3.2 광고주 기능

  • [필수] 미션 등록(제목, 설명, 예산, 마감일, 브랜드 에셋 업로드, IP 활용 옵션)
  • [필수] 미션 관리(수정 / 마감 / 삭제)
  • [필수] 시안 목록/상세(영상 재생)/선정
  • [필수] 결제 요청(선정 시 결제 링크 발행 → 입금 확인 후 미션 활성화)

3.3 크리에이터 기능

  • [필수] 미션 목록(검색·카테고리 필터·정렬)
  • [필수] 미션 상세(활용 가능 IP 정보 표시)
  • [필수] 시안 제출(제목·설명·영상 업로드 200MB 이내·사용 IP 명시·첨부파일)
  • [필수] 내 제출 내역 / 선정 결과 확인

3.4 관리자 페이지

  • [필수] 회원·미션·시안·IP 조회 및 수정
  • [필수] 시드 데이터 입력 도구(IP, 가짜 미션·시안)
  • [필수] 신고 / 차단 처리
  • [필수] 결제 입금 확인 및 미션 활성화 처리
  • [필수] Slack / 이메일 알림(신규 미션 등록 · 시안 제출)
  • [필수] 통계 대시보드(가입자 / 미션 / 시안 / 활성 사용자)

3.5 IP Showcase

  • [필수] IP 카탈로그 페이지(이름, 썸네일, 설명, 사용 조건)
  • [필수] 미션 등록 시 IP 연결
  • [필수] 시안 IP 표기 · 뱃지
  • [필수] 관리자 수동 등록 · 관리

3.6 결제 링크

  • [필수] 토스페이먼츠 결제 링크 또는 무통장입금 안내
  • [필수] 관리자 수동 입금 확인 및 미션 활성화
